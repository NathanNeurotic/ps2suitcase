# PS2 Rust

Monorepo of my Rust projects for PS2 homebrew.

## Crates

### ps2-filetypes

A collection of PS2 file type parsers.

### ps2-mcm

Memory Card Manager

### psu-packer-gui

Graphical interface for packing PSU archives. Run with:

```
cargo run -p psu-packer-gui
```
